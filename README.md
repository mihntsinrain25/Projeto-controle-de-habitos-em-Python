# Projeto-controle-de-habitos-em-Python
Projeto realizado na matéria de "Algoritmo e Estrutura de Dados I" da Fatec de Votorantim. O projeto escolhido se deve ao fato de que muitas pessoas querem ser mais organizadas, mas não sabem por onde começar. Por isso programa funciona como uma ferramenta prática para listar hábitos importantes e ter uma visão básica da própria rotina.
